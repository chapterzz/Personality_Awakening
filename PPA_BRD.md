# 业务需求文档 (BRD) - 性格星球：觉醒计划 (v4.0)

## 1. 项目愿景
构建一个**云端互联、实时同步、科学严谨**的人格探索平台，打破传统本地测试的数据孤岛。让初中生能在任何设备上通过“探险”了解自我，让老师能实时掌握班级心理图谱，成为学校心理健康教育的现代化基础设施。

## 2. 核心业务目标
1.  **去孤岛化**：实现学生端（手机/PC）与教师端（大屏/PC）数据**低延迟、近实时**同步，彻底消除文件导入导出流程。（技术指标以 NFR「端到端 &lt; 1s」为准；可选单独观测 WebSocket 通道层延迟作为内部指标。）
2.  **游戏化体验**：通过 AVG（文字冒险）模式将枯燥测试转化为“性格星球探险”，提升用户完成率至 95% 以上。
3.  **隐私合规**：严格遵循未成年人保护原则，采用“昵称 + 随机码”匿名机制，不收集真实身份信息，提供一键销毁功能。
4.  **科学普及**：在娱乐中嵌入 MBTI 科学边界科普，防止“标签效应”，引导正确自我认知。

## 3. 用户角色与场景
| 角色 | 典型场景 | 核心诉求 |
| :--- | :--- | :--- |
| **学生 (Explorer)** | 在家用手机/PC完成测试，查看精灵形象，分享海报。 | 有趣、好看、隐私安全、能跨设备保存进度。 |
| **教师 (Guide)** | 创建班级，发送邀请码，课上实时投屏班级性格分布图。 | 操作极简、数据实时刷新、可视化直观、有教学建议。 |
| **管理员 (Admin)** | 监控系统运行，管理内容库（题目、精灵文案）。 | 系统稳定、内容可配置、数据安全。 |

## 4. 业务范围 (Scope)
### 4.1 包含功能 (In-Scope)
- **用户系统**：游客模式、极简注册（昵称/密码）、临时数据迁移、个人中心；**支持先注册再测评**，进行中进度云端保存，登录后续答（详见 PRD）。
- **测试引擎**：标准模式（93 题简化版）、AVG 冒险模式（4 章节剧情）、自适应题库逻辑。
- **班级系统**：创建班级、6 位加入码、实时成员统计、权限控制；**教师与学生在同一用户体系中注册**，通过角色区分，**班级创建者即该班 `teacher_id`（关联 `User`）**（详见 PRD）。
- **数据可视化**：学生端个人报告（雷达图/精灵卡）、教师端班级热力图（实时 WebSocket 推送）。
- **科普图书馆**：MBTI 百科、反标签专题、名家案例。
- **社交分享**：生成专属海报（Canvas）、二维码语音彩蛋。

### 4.2 排除功能 (Out-of-Scope)
- **离线模式**：必须联网使用，不处理本地持久化存储（除短暂缓存外）。
- **真实身份认证**：不接入手机号实名验证，不收集身份证号。
- **复杂社交网络**：无好友系统、无公开评论区，仅限班级内部匿名聚合数据。
- **原生 App**：仅开发 PWA/Web 应用，不打包 iOS/Android 原生包。

## 5. 非功能性需求 (NFR)
- **并发性能**：支持单班级 50-60 人同时在线测试，全校多班级并发访问（目标 QPS: 200+）。
- **实时性**：端到端实时延时（学生提交 -> 教师可见）< 1 秒。
- **响应速度**：首屏加载 < 1.5s (SSR)，交互响应 < 100ms。
- **兼容性**：完美适配移动端 (iOS Safari, Android Chrome) 和桌面端。
- **安全性**：全站 HTTPS，密码加盐哈希，防 SQL 注入，XSS 过滤，敏感操作日志记录。
- **数据保留**：用户注销后，物理删除所有个人关联数据（GDPR 风格）。

## 6. 成功指标
- 学生测试完成率 > 90%。
- 班级创建后，学生加入率 > 85%。
- 教师端无需任何手动刷新即可看到最新数据。
- 零重大安全事故，零隐私泄露投诉。
