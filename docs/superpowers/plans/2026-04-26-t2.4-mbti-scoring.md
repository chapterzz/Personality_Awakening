# T2.4 MBTI 评分算法服务 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `apps/api` 交付 MBTI 评分内部模块与 `POST /scoring/mbti` API，支持 `STANDARD`/`AVG` 信号输入，并通过单元+集成测试验证。

**Architecture:** 新增 `scoring` 模块，controller 负责 HTTP 校验与响应格式，service 负责纯计算与最小一致性检查。算法输入统一为 `signals`，避免耦合题库/CMS。输出包含四维分值、winner、delta 与 `mbti_type`。

**Tech Stack:** NestJS 10、class-validator、Jest、Supertest、TypeScript strict
