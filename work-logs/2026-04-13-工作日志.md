### 遇到问题 & 解决方案

| 问题 | 处理 |
| :--- | :--- |
| `git push` 连接被重置 | 在用户本机网络下重试 push；按需开 VPN、配置代理或改用 SSH。 |
| Playwright 需同时起 web + api | 使用双 **`webServer`** + api **`serve:e2e`**，避免仅起前端导致 API E2E 失败。 |
| 本环境无 Docker，无法自动验证迁移落库 | 迁移 SQL 已入库；你在本机起 Postgres 后执行 **`pnpm db:migrate`**，可用 **`pnpm exec prisma studio`** 目视表结构。 |
| `pnpm install` / Prisma 引擎拉取较慢 | 属网络耗时；安装完成后 **`prepare`** 内 **`prisma generate`** 已能正常完成。 |
