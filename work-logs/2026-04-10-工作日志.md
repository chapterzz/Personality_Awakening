### 遇到问题 & 解决方案

| 问题 | 处理 |
| :--- | :--- |
| 首次 `create-next-app` 报路径不可写 | 先创建 `apps/` 目录后重试成功。 |
| `pnpm approve-builds` 为交互式，阻塞 CI/脚本 | 在根 `package.json` 增加 `pnpm.onlyBuiltDependencies` 白名单。 |
| `@nestjs/swagger@11` 与 Nest 10 peer 不匹配 | 降级为 `@nestjs/swagger@7.4.2`。 |
| `next build` 报缺少 `eslint-import-resolver-typescript` | `pnpm --filter web add -D eslint-import-resolver-typescript`。 |
| 根目录 `prepare` / Husky 提示找不到 `.git` | 当前环境未配置 Git 或 PATH 无 `git`；钩子文件已就位，待 `git init` 并安装 Git 后生效。 |
